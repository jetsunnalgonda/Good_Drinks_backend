Welcome to Good Drinks API

This API provides information about various drinks and their recipes.

API Endpoints:

  • GET /api/drinks: Retrieve all drinks
  • GET /api/drinks/{drink_id}: Retrieve a specific drink by ID
  • GET /api/drinks/random: Retrieve a random drink
  • POST /api/drinks: Create a new drink
  • PUT /api/drinks/{drink_id}: Update an existing drink
  • DELETE /api/drinks/{drink_id}: Delete a drink

Enjoy exploring the world of Good Drinks!
